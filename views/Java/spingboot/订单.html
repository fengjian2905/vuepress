                            # 取舍问题

                            99.87 * 0.85 = 84.8895

                            保留两位：

                            平台赚钱：向上取整 进一 89.8895 = 89.89
                            平台亏钱：直接舍去 89.88
                            四舍五入：有赚有亏
                            四舍六入：重点在对于5的处理

                            绝对精确：平衡 舍去 多赚 平衡（四舍五入达不到，四舍六入可以）

                            对于平台来说亏多少赚多少取决的因素：
                            成交额 1年成交金额 10亿 1笔 订单数量 1年的订单数量 10亿笔 1块一笔

                            银行家算法

                            四舍六入五考虑，五后非空就进一，五后为空看奇偶，五前为偶应舍去，五前为 奇要进一

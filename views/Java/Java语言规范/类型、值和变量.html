    风间 2020-03-31 类型 变量 Java

    Java是静态类强型语言，每个变量和表达式在编译时就必须有明确的类型， 并且限制了变量和表达式的值，有助于在编译时检测错误

    # 简单类型(PrimitiveType)

    简单类型值之间不共享状态

    # 数字类型(NumericType)

    1. 整数：byte（8），short（16），int（32），long（64），char（16）

    常用值：
    byte类型（-128~127闭区间），shrt类型（-32768~32767闭区间）， char（0~65535闭区间）

    除移位操作符外的整数操作符，有一个操作数是long类型的，那么操作就会 按照64位精度操作（就是会将不是long类型的数字转为long类型），结果也是long类型；
    否则，按照32位精度执行，结果也是int类型的；

    数值x 数值y 运算时 返回值
    long类型 非long类型 将y转换为long类型进行运算 long类型
    int类型 非int类型 将y转换为int类型进行运算 int类型
    非int类型 非int类型 将x和y转换为int类型进行运算 int类型
       byte a = 3;
   byte b = 4;
   System.out.println(Integer.class.isInstance(a));//false
   System.out.println(Integer.class.isInstance(b));//false
   System.out.println(Integer.class.isInstance(a+b));//true
   
   long a = 3;
   int b = 4;
   System.out.println(Integer.class.isInstance(a));//false
   System.out.println(Integer.class.isInstance(b));//true
   System.out.println(Integer.class.isInstance(a+b));//false
   System.out.println(Long.class.isInstance(a+b));//true

    1. 浮点数：float（32），double（64）

    浮点数包含：正负0、正负无穷、NaN（Not a Number）, NaN表示某些无效操作的结果，例如0除0
    float和double的NaN常量被预定义为Float.NaN和Double.NaN

       System.out.println(0.0==-0.0);//true
   System.out.println(0.0/0.0);//NaN
   System.out.println(1.00/-0.0);//-Infinity
   System.out.println(1.00/0.0);//Infinity

    除NaN外，读点书都是有序的

    操作数（x、y） 操作符 返回结果
    1个或2个NaN <、<=、>、>= false
    1个NaN == false，(x<y)==!(x>-y)为false
    1个NaN != true,当且仅当x为NaN时，x!=x为true
    
   float x = Float.NaN;
   float y = 3;
   System.out.println(x<y);//false
   System.out.println(x>-y);//false
   System.out.println((x<y)==!(x>-y));//false
   System.out.println(x!=x);//true

    数值x 数值y 运算时 返回值
    double类型 非double类型 将y转换为double类型进行运算 double类型
    float类型 非float类型 将y转换为float类型进行运算 float类型

    浮点类型和其他数字类型可以进行双向转换，浮点类型和boolean型之间是无法转换的

    Java在将浮点值向整数转换时，会使用向0舍入的模式，即会将数字截尾，丢掉尾部的部分

    非规格化数和渐进下溢
    上溢的浮点操作会产生一个有符号的无穷
    下溢的浮点操作会产生一个非规格化的值或有符号的0
    结果没有书序定义的浮点操作会产生一个NaN
    操作数中有NaN的所以数值操作会产生一个NaN作为结果

    # 布尔类型(boolean)

    boolean类型取值：字面常量true，false
    整数和浮点数转换为boolean类型：通过x!=0的方式实现
    引用对象obj可以通过obj!=null来转换为boolean类型
    以上都是遵循了C语言规范：非0为true、非null为true

    1. 对象

    a. 对象是类的实例或数组

    b 引用值（引用）是指向这些对象的指针，并且有一个特殊的空引用，它不指向任何对象。

    c. 绝大多数对象都具有状态，这些状态存储在对象的域中， 如果两个变量包含指向同一个对象的引用，那么通过一个变量的对象的引用修改该对象的 状态时，通过另一个变量中的引用可以观察到变化后的状态

       int a = 3;
   int b = a;
   b = 4;

   Demo c = new Demo();
   c.setName("c");
   Demo d = c;
   d.setName("d");
   System.out.println(a);
   System.out.println(b);

   System.out.println(d.getName());

    1. Object类 Object类是所有其他类的超类
      所有类和数组类型都继承了Object类的方法，具体包括： clone方法，用来创建对象的副本
      equal方法，定义了对象判等的方法，该标准应该是基于值比较而不是引用比较的
      finalize方法，在对象销毁前执行
      getClass方法，返回表示对象所属类的Class对象
      hashCode方法
      wait、notify和notifyAll方法，在使用线程的并发编程中会用到
      toString方法，返回对象的String表示

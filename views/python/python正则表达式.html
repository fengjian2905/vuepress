# 字符集[]

[cf]是字符是c或者f就匹配，[c-f]字符是c到f中的一个就匹配
找出中间字母是'c'或者'f'的字符串
普通字符通常用来定界

    import re
    s = 'abc,acc,adc,aec,afc,ahc'
    r = re.findall('a[cf]c',s)
    print(r)

# 概括字符集

不管是字符集还是概括字符集都只能匹配单一字符
'\d':数字
'\D':非数字
'\w':单词字符
'\W':非单词字符（包括空格、回车、制表符...）
'\s':空白字符['','\t','\r','\n']
'\S':非空白字符
'.':匹配除'\n'行符外的所有字符

# 数量词{n}

'[a-z]{3,6}'
贪婪匹配：如果3个和6个都满足都情况，会取6的匹配结果， python尽可能多的匹配
'[a-z]{3,6}?'
?将贪婪模式转非贪婪， 非贪婪匹配：如果3个和6个都满足，则取3都匹配结果

# re.sub函数：字符串替换

re.sub(a,b,c,d,e)
a:正则表达式，筛选出要替换的字符
b:用来替换的字符
language：原始字符串
d:默认0，匹配到的所有字符都被替换，1只替换到匹配到的第一个

    import re
    language = 'PythonC#\nJavaPHP'
    r = re.sub('C#','GO'，language,0)
    # language.replace('C#','GO') 由于字符串不变性，并不会实现目的
    print(r)

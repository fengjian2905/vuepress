    # 集合 set

    序列是有序的，集合是无序的

    • 集合的定义使用{}
      >>> type({1,2,3,4,5,6})
  <class 'set'>
    1
    2
    • 集合是无序的，不能使用序号获取元素，也不能使用切片操作
       >>> {1,2,3,4}[1]
   Traceback (most recent call last):
     File "<stdin>", line 1, in <module>
   TypeError: 'set' object does not support indexing
    1
    2
    3
    4
    • 集合中的元素是不重复的
       >>> {1,1,2,2,3,3}
   {1, 2, 3}
    1
    2
    • 集合是支持长度判断的
       >>> len({1,2,3})
   3
   >>>
    1
    2
    3
    • 集合是否包含某个元素
       >>> 1 in {1,2,3}
   True
   >>> 1 not in {1,2,3}
   False
    1
    2
    3
    4
    • 集合的特有操作 两个集合的'差集'
       >>> {1,2,3,4,5,6}-{3,4}

    两个集合的交集

       >>> {1,2,3,4}&{1,2}
   {1, 2}

    2

    两个集合的并集（合并、不重复）

       >>> {1,3,4}|{3,4,5}

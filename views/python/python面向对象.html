                      # 类的使用

                      1. 类最基本作用：封装
                      2. 类的使用和类的定义不要写在一个模块中

                      # 方法和函数的区别

                      1. c、c++中 函数，java、c#中 方法
                      2. 方法：设计层面，函数：程序运行、过程式的一种称谓
                      3. 定义在类中的函数叫方法，定义在模块中的叫函数
                      4. 变量出现在模块中就叫做变量，出现在类中叫做数据成员
                      5. 没有必要特意强调方法和函数的区别

                      # 类和对象到底是什么

                      1. 类是现实世界或思维世界中的实体在计算机中的反映, 他将数据以及这些数据上的 操作封装在一起
                      2. 行为与特征：类的数据成员（变量）来表示对象的特征， 方法（函数）用来表示对象的行为
                      3. 行为没有找对主体是面向对象设计不合格的最大原因
                      4. 对象是具体，类是抽象
                      5. 类是模版，是印刷机，通过这个类可以产生多个对象

                      # 类变量和实例变量

                      1. 类变量是和类相关联的变量
                      2. 实例变量是和对象相关联的变量,使用self.xxx定义实例变量
                      3. 访问实例变量，如果没有找到会到类变量中寻找同名的变量，类 变量也找不到会到父类变量中找

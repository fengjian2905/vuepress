      # 分页加载数据细节

      1.如果服务器一条数据都没有 空
      2.最后一页，还有没有更多都数据
      3.累加，服务器有100条数据，第一次1-20 第二次21-40 ...，setData重新渲染所有数据
      4.非分页数据加载状态：正在加载 数据为空；
      分页数据加载状态：正在加载 加载完成 有更多数据（提醒用户不要在滑滑滑了）
      5.上滑页面触底，避免用户重复发送请求
      6.按钮点击操作：防抖 截流 （禁用、倒计时、模态loading、数据锁...）
      7.start count,每次请求start需要重新计算
      8.封装性（类-ES6、函数-ES5）

      # 定义Paging类封装分页加载数据细节

      美好的愿望：
      当我们要获取分页数据的时候，不需要关心细节，我们就告诉Paging '我需要分页数据了，你能给我吗？'， 由Paging对象决定是否返回我们所需要的数据，Paging对象负责跟服务器交互。

      # 初实现

      1.Paging对象需要一个公共的方法，供外部调用getMoreData用于获取下一页数据
      2.在Paging的构造函数中设置服务器接口所需要的参数constructor(url,count,start)，初始化加载数据
      3.getMoreData向服务器发送请求时，使用锁（loker）来避免用户重复发送请求，每次请求之前检测锁是否锁住(getLocker)， 锁住的情况下不允许发送请求，一次请求结束后要释放锁(releaseLocker)

      class Paging {

    start
    count
    url
    locker = false

    constructor(url,count=10,start=0){
        this.url = url
        this.count = count
        this.start = start
    }

    getMoreData(){
        //getLocker
        //request
        //releaseLocker
        if (!this._getLocker()) {
                    return
        }
        this._actualGetData()
        this._releaseLocker()
    }

    _actualGetData(){
        Http.request()
    }

    _getLocker(){
        if (this.locker) {
            return false
        }
        this.locker = true
        return true
    }
    
    _releaseLocker(){
        this.locker = false
    }
}

      # 再次加工

      1.Http的request方法所需要的参数request({url, data, method = 'GET'})，这里request是接收一个对象， 这个对象表明用户还可能传递data和method参数，构造函数中只是有一个url不能满足需要， 所以constructor的参数要把url改为一个req，这个req（对象）可以包含url、data、method
      2.req中的url的形式v1/spu/latest?start=0&count=10，这里不可以直接使用 Http.request(req)，req中的url还需要进行处理，start和count的拼接（不需要用户来处理）， _getCurrentReq

          class Paging {
    
        start
        count
        req
        locker = false
        url
        constructor(req,count=10,start=0){
            this.req = req
            this.count = count
            this.start = start
            this.url = req.url
        }
    
        getMoreData(){
            if (!this._getLocker()) {
                return
            }
            this._actualGetData()
            this._releaseLocker()
        }
    
        _actualGetData(){
            Http.request(this.req)
        }
    
        _getCurrentReq(){
            //let url = this.req.url 如果这样定义，
            // 在下面代码this.req.url = url时会出现重复拼接错误，
            //所以在属性中添加一个url来保存req中的初始url
            let url = this.url
            const params = `start=${this.start}&count=${this.count}`
            if (url.includes('?')) {
                url +='&'+params
            }else{
                url +='?'+params
            }
            this.req.url = url
            return this.req
        }
    
        _getLocker(){
            if (this.locker) {
                return false
            }
            this.locker = true
            return true
        }
    
        _releaseLocker(){
            this.locker = false
        }
    }

      # 处理服务端返回数据

      Paging对象不可能直接把服务器返回的数据原封不动的返回给用户，还要针对上面分析的细节 进行处理，定义一个对象将以上细节返回给用户是比较合适的
      这个对象包含的属性：
      empty（一条数据都没有）
      items（分页数据，当前请求的数据）
      moreData（是否最后一页）
      accumulator(累加器，历史请求的所有分页数据)

         {
       empty:boolean,
       items:[],
       moreData:boolean,
       accumulator:[]
   }

      # 最终版本

      class Paging {

    start
    count
    req
    locker = false
    url
    moreData
    accumulator

    constructor(req,count=10,start=0){
        this.req = req
        this.count = count
        this.start = start
        this.url = req.url
    }

    async getMoreData(){
        if (!this.moreData){
            return
        }
        if (!this._getLocker()) {
            return
        }
        const data = await  this._actualGetData()
        this._releaseLocker()
        return data
    }

    async _actualGetData(){
        const req = this._getCurrentReq()
        let paging = await Http.request(req)
        if (!paging) {
            return
        }
        if (paging.total===0){
            return {
                empty:true,
                items:[],
                moreData:false,
                accumulator:[]
            }
        }
        this.moreData = Paging._moreData(paging.total_page,paging.page)
        if (this.moreData){
            this.start += this.count
        }
        this._accumulator(paging.items)
        return {
            empty: false,
            items: paging.items,
            moreData: this.moreData,
            accumulator: this.accumulator
        }

    }

    _accumulator(items){
        this.accumulator = this.accumulator.concat(items)
    }

    static _moreData(totalPage,pageNum){
        return pageNum < totalPage -1
    }

    _getCurrentReq(){
        let url = this.url
        const params = `start=${this.start}&count=${this.count}`
        if (url.includes('?')) {
            url +='&'+params
        }else{
            url +='?'+params
        }
        this.req.url = url
        return this.req
    }

    _getLocker(){
        if (this.locker) {
            return false
        }
        this.locker = true
        return true
    }

    _releaseLocker(){
        this.locker = false
    }
}
